# Weather Data Analysis using Hadoop, MapReduce, Pig, and Hive

## Project Overview
Developed a Big Data analytics project using the Hadoop ecosystem to process and analyze large-scale weather datasets. Built a multi-node Hadoop cluster using team members' Virtual Machines and performed distributed data processing using HDFS, MapReduce, Apache Pig, and Apache Hive. The project demonstrates distributed computing, data cleaning, data aggregation, and large-scale analytics in a clustered Hadoop environment.

## Project Objective
To analyze large-scale weather datasets using the Hadoop ecosystem and compare different Big Data processing approaches, including MapReduce, Apache Pig, and Apache Hive.

## Cluster Setup
- Configured a multi-node Hadoop cluster using team members' Virtual Machines.
- Implemented distributed storage using Hadoop Distributed File System (HDFS).
- Executed distributed processing and analytics across cluster nodes.
- Enabled scalable data storage and computation within the Hadoop ecosystem.

## Dataset
The project uses weather data containing:
- Temperature records
- Precipitation data
- Date and location information
- City and state details
- Wind-related attributes

## Data Processing Workflow
1. Data Ingestion into HDFS
2. Data Cleaning using MapReduce
3. Weather Analysis using MapReduce
4. Data Exploration using Apache Pig
5. Query-Based Analytics using Apache Hive
6. Result Generation and Interpretation
